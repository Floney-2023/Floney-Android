## 주요 성과 🛠
* 앱 전체 구조를 MVVM + Clean Architecture로 설계하고, 모듈 분리와 Hilt 기반 의존성 주입 설정을 주도함
* 다크모드 대응, 이미지 캐싱 최적화(Glide), 화면별 UI QA를 진행하며 사용자 경험 개선
* 가계부 생성, 내역 추가, 회원가입/로그인 등 주요 기능 구현 및 화면 연결
* 기획자와의 꾸준한 의사소통을 통해 QA 및 릴리즈 전 최종 기능 점검 주도
* 구독제 정기 결제 기능 개발 진행 중

<br/>

## 💻 기술 스택

* Tool : Android Studio

* Architecture : MVVM, Clean Architecture

* Network : Retrofit, OkHttp

* JetPack Components : LifeCycle, ViewModel, EventFlow, StateFlow, DataBinding, DataStore

* Dependency Injection : Hilt

* Communication Tool : Notion, Slack, Figma, Postman
